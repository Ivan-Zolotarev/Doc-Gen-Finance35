# Запуск приложения для технической документации

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск веб-приложения

```bash
python app_tech.py
```

### 3. Откройте в браузере

```
http://127.0.0.1:5000
```

## Что вы увидите

## Примеры использования

### Пример 1: Документация вашего проекта

1. Создайте ZIP архив вашего проекта
2. Перейдите на `/code`
3. Загрузите архив
4. Скачайте сгенерированную документацию

### Пример 2: API документация

1. Перейдите на `/api`
2. Загрузите файл с Flask приложением (например, `app.py`)
3. Выберите формат (Markdown или OpenAPI)
4. Скачайте документацию

### Пример 3: Документация БД

1. Перейдите на `/database`
2. Загрузите SQL файл со схемой
3. Выберите формат (Markdown или Mermaid диаграмма)
4. Скачайте результат

## Решение проблем

### Ошибка импорта модулей
```bash
# Убедитесь, что вы в корневой директории проекта
cd Doc-Gen-Finance35
python app_tech.py
```

### Ошибка при анализе кода
- Убедитесь, что файл содержит валидный Python код
- Проверьте, что файл имеет расширение `.py`

### Ошибка при анализе БД
- Для SQL файлов: убедитесь, что файл содержит CREATE TABLE команды
- Для подключений: проверьте строку подключения и доступность БД
